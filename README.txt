Приложение Todo List
Это консольное приложение для управления списком дел, написанное на Go. Оно позволяет пользователям добавлять, просматривать, редактировать, завершать и удалять задачи. Также поддерживает установку приоритета задач и фильтрацию по статусу.

Функции
Добавление задачи: Позволяет добавлять задачи с заголовком и описанием.
Просмотр всех задач: Отображает список всех задач в табличном виде.
Редактирование задачи: Позволяет обновлять заголовок и описание существующих задач.
Изменение статуса задачи: Помечает задачи как выполненные или невыполненные.
Удаление задачи: Удаляет задачи из списка.
Установка приоритета задачи: Присваивает задаче уровень приоритета (1-5).
Фильтрация задач: Фильтрует задачи по статусу (выполнены/невыполнены).
Сортировка задач: Сортирует задачи по дате, статусу или приоритету.

Использование
Добавление задачи: Выберите опцию 1 в меню для добавления новой задачи.
Просмотр задач: Выберите опцию 2, чтобы просмотреть все задачи.
Редактирование задачи: Выберите опцию 3 для редактирования задачи.
Изменение статуса задачи: Выберите опцию 4 для изменения статуса задачи.
Удаление задачи: Выберите опцию 5 для удаления задачи.
Установка приоритета задачи: Выберите опцию 6 для установки приоритета задачи.
Фильтрация задач: Выберите опцию 7, чтобы отфильтровать задачи по статусу.
Сортировка задач: Выберите опцию 8, чтобы отсортировать задачи по дате, статусу или приоритету.
